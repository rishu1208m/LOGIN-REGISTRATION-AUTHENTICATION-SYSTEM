# LOGIN-REGISTRATION-AUTHENTICATION-SYSTEM
A full-stack authentication system built using React, Node.js, Express, and MongoDB. The application supports secure user signup and login using JWT authentication, password hashing with bcrypt, and protected routes. Designed to demonstrate real-world authentication flow and backend security practices.
# Authentication System (Full Stack Project)

## Features
- User Signup & Login
- JWT Authentication
- Protected Routes
- Password Hashing using bcrypt

## Tech Stack
- React.js
- Node.js + Express
- MongoDB

## How to Run

### Backend
cd server  
npm install  
npm start  

### Frontend
cd client  
npm install  
npm start
